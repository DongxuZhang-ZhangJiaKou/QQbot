# QQbot

#### 使用方法

第一步：

```
git clone https://github.com/DongxuZhang-ZhangJiaKou/QQbot.git
```

将项目克隆下来

第二步：

在终端执行python3 main.py

第三步：

打开先驱，当出现

```
GET /ws/ 1.1 101 - 96153
```

恭喜你，你成功了，剩下你就可以去自由发挥了

#### 主要功能

###### 腐竹利器

你可以直接在聊天框输入”服务器信息“

###### 聊天能手

你可以直接在聊天框里聊天

注：在群里要加上小艾

###### 天气预报

你可以在聊天栏输入”天气[空格]地名“

或者直接输入”天气也可以“
